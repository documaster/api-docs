Code Lists

Overview

Code Lists are configurable set of values used as predefined values for entity fields. Below is the complete list of code lists. Some of the code lists come with preconfigured values and some of them are empty as being dependent on customer requirements.

Code Lists

Code Name Comment

A Avsluttet

Code Name Comment
